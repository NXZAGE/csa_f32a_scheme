# csa_f32a_scheme

## Примечания
  - **RZ** - zero флаг для R
  - Для всех регистров есть соответствующие latch
  - Для всех mux есть соответсвующие CS
  - Стеки R и D реализованы идентично и изображены на схеме упрощенно (2 входа и 2 выхода на каждом регистре). Stack Register View показывает более строгую версию. На mux всех регистров DataStack приходят соответствующие (единые) сигналы push_D или pop_D. Для RetStack аналогично.
  - S T и R имеют собственые mux cs сигналы
